MILKY WAY

AIRWAYS BANK
Hall Reverb
'Cabin fever' knob defines the decay of the reverb or hall size. Holding 'tap' for longer than 1 second enables the secondary function: fixed hi-pass filter to cut off low frequencies and have more 'air' in the final output.
Shimmer Reverb
Variation of the hall reverb with a pitch shifter to create choir-like spaces. Primary 'cabin fever' defines decay, secondary function defines the amount of pitch-shifter mixed into original reverb.
Stereo Room Reverb
Recreates a sort of stereo room ambience. Primary 'cabin fever' defines room size and the secondary defines the stereo spread of the reverb, from mono up to a huge stereo spread.
Plate Reverb
Primary 'cabin fever' defines the decay of the reverb (distance from pickups to metal plate). Secondary parameter defines the amount of pre-delay to distant sounds in ambience.
Spring Reverb
Primary 'cabin fever' defines decay. With 'tap' button you can simulate plucking the real spring. Secondary function is tied to 'pluck the spring' feature and defines how fast the spring will calm down.
Ping-Pong Delay
A stereo clocked delay. A 'tap' is 3+ short clicks. Primary 'cabin fever' defines feedback. Secondary defines clock division of incoming tap/clock: 1, 3/4, 2/3, 1/2, 1/3, 1/4, 1/8.
Tape Echo
Delay with 3 fixed playback heads. Primary 'cabin fever' defines repeat rate (speed of tape). 'Tap' defines feedback. Secondary works as a divider for the incoming clock.
Chorus
Primary 'cabin fever' defines feedback amount (up to infinite unrealistic ambient). Secondary parameter defines modulation depth, which is 'full on' by default.
DARKWAVES BANK
Gated Reverb
Plate reverb with noise gate. Primary 'cabin fever' defines reverb decay, secondary defines threshold of the noise gate. Attack and decay are fixed.
Spring Reverb
Primary 'cabin fever' defines decay. With 'tap' button you can simulate plucking the spring. Secondary function defines DECAY of how fast the spring will calm down after manually plucking.
Reversed Reverb
Takes reverb tail and reverses it (breathing effect). 'Cabin pressure' defines predelay (dry/wet). 'Cabin fever' sets decay. Secondary function: damping (volume of the tail/head).
Flanger
'Cabin pressure' sets amount of delay. Primary 'cabin fever' sets LFO speed. Secondary defines feedback for sweeping, airplane engine alike sounds.
Ring Modulator
Multiplies signal with internal sine wave. 'Cabin pressure' defines modulation amount, 'cabin fever' defines speed. Secondary function controls feedback for special dirtiness.
Overdrive
'Cabin pressure' adjusts drive with volume compensation, 'cabin fever' defines tone control. 'Tap' button makes the effect active/bypassed (latching via trigger CV input).
Peak Compressor
'Cabin pressure' defines threshold (-90dB to 0dB). Primary 'cabin fever' sets gain reduction ratio. Secondary defines attack (1-200ms). Release is 'auto'. CV input acts as side-chain.
Freezer / Looper
When 'tap' is pressed (or CV gate is ON), audio is looped by the grain length defined by 'cabin fever', with speed defined by 'cabin pressure'.

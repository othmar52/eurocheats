# EuroCheats 🎛️

**EuroCheats** is an open-source collection of ultra-compact, visual cheat sheets for Eurorack modules. 

If you've ever returned to a module after a few weeks and completely forgotten how to access that one specific hidden mode, or what the 16 different LED colors actually mean, this repository may be for you.

## The Philosophy

* **Essentials Only:** This is *not* a replacement for the official manuals. There is no deep-dive patching philosophy here. EuroCheats provides only the necessary fragments: button combos, scale lists, and algorithm overviews. 
* **Mobile First:** The cheat sheets are built with simple HTML/CSS and are highly responsive. They look like native dark-mode apps on your smartphone, making them perfect for a quick glance while standing in front of your rack.
* **The Screen-Free Vision (Printable):** Eurorack is a tactile, screen-free experience, and we want to keep it that way. Every cheat sheet ~is~ will be heavily optimized for printing via CSS (`@media print`). The long-term goal of this project is to allow users to print these out, cut them into handy physical cards, laminate them, and keep them right next to the patch cables—completely smartdevice-free.

## Features

* **Visual Button Indicators:** Custom SVG and CSS graphics that mimic the actual hardware (e.g., Make Noise button combos, Robaux LED arcs).
* **Responsive Grid:** Automatically switches from a single-column mobile view to a dense, multi-column layout on desktop/landscape.
* **TODO: Print-Optimized:** When printed, the dark themes automatically invert to black-on-white to save ink, and interactive UI elements (like toggle buttons) are hidden.
